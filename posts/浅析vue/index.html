Vue的两个版本

vue有两个版本，分别是完整版（Vue.js）和非完整版（vue.runtime.js），两者的区别：

两个版本推荐使用非完整版，然后配合vue-loader和vue文件使用。

template 和 render 怎么用

用template——html的方式做渲染

完整版的vue必须使用template。

用创建一个加1的代码举例：

new Vue({
  data: {
    n: 0,
  },
  template: `
  <div id="app">
  <p>{{ n }}</p>
  <button @click="add">+1</button>
  </div>
  `,
  methods: {
    add() {
      this.n += 1;
    },
  },
}).$mount('#app');

render——js的方式做渲染

非完整版需要用render写法，用 const h = creatElement创建一个标签。 用同样的代码举例：

new Vue({
  el:'#app',
  data:{
    n:0,
  },
  
  //用js创建一段HTML
  render(h){
    return h('div',[
      this.n,
      h(
        'button',
        {
          on:{
            click:this.add, 
             },
          },
          '+1',
        ),
    ]);
  },
  methods:{
    add(){
      this.n +=1;
    },
  },
});
